# Hotel Analytics Api #

A python RESTful API service that serves data for the Hotel Analytics product.

#### Project Setup ####

1. Clone this repo.
2. Change directory to repo.
3. Setup the dev environment.
	* Setup Docker (windows users see requirements below)
	* Install Python Fabric (2.x Python version requirement)
4. Build the docker container by running:      ```fab build```
5. Start and connect to the container by running:      ```fab dev```
6. Once connected you can start the service by running:      ```python run.py```
